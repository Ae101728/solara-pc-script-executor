## FAQ

**Is Solara a local desktop tool?**  
Yes. It is described as a Windows desktop application with local operation and built-in script management.

**Will it stay compatible after Roblox updates?**  
The auto-update engine is meant to help the build stay aligned with newer releases, though compatibility can still change over time.

**How is it different from a basic script loader?**  
Solara includes a script hub, persistent queue storage, batch execution, and a debugger instead of only a simple launch path.

**Does using a script executor affect account risk?**  
Third-party tools used with Roblox may carry account and platform risk. Check Roblox terms and use your own judgment before running any script.

**Does Solara store anything locally?**  
Yes. The persistent queue relies on SQLite storage, and related settings or hub data may remain on the device for later sessions.
